Replace existing images or add to existing images with this upload?

Use the following naming convention so that the images are uploaded in preview / full image pairs that you expect!

[two digit number][dash][P or F][double underscore][image name]

Use the numbers to pair two imagest together and the P or F to designate preview image or the full image

For example these two files would be paired, with the first as the preview image and the second as the full image:

03-P__myPosterSmall.png and 03-F__myPoster.png

Max upload is 100 images (50 preview / 50 full). All images must be type .png and less than 3mb
