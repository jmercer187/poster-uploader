Upload the preview and full images below. In order that the preview images and the full size images match, use the same prefix for each followed by a '__' double underscore.

For example loading this into Preview Images:

1__aNicePreviewImg.png

would correspond to this loaded in Full Images:

1__aNiceFullSizeImg.png

Use what you want as the identifying prefix .. just make sure the double underscore '__' is at the end of the identifying prefix, before the actual name of the image.

All images must be type .png and less than 3mb
